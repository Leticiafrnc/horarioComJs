## Hora do Dia 
Esse projeto foi desenvolvido à partir dos ensinamentos do professor [Gustavo Guanabara - no Curso em video de JavaScript](https://www.youtube.com/watch?v=BXqUH86F-kA&list=PLntvgXM11X6pi7mW0O4ZmfUI1xDSIbmTm) foi criado para fortalecer meus aprendizados da aula de DOM e condições compostas e aninhadas no JavaScript. 

<h4 align="center"> 
🚀 Finalizado
</h4>

### 📕 Aprendizados
- Aprendi mais sobre condições aninhadas no javascript
- Sobre o método getHours() para retornar a hora de acordo com sistema 
- Colocar a imagem com img.src ='nomedaimagem.jpg' pelo javascript
- Pratiquei também as media query no Css

### 🏁 Instalação

Antes de começar, é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)
Clone este repositório usando git clone: https://github.com/Leticiafrnc/horarioComJs.git ou clique em Code/Download ZIP. Clique no arquivo "index.HTML" e veja a aplicação.
